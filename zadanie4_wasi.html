Zadanie 4 - Odpowiedzi

1) Kiedy powstał JavaScript?

JavaScript powstał w grudniu 1995 roku.

2) Dlaczego jest to język skryptowy?

Jest to język skryptowy, ponieważ jest interpretowany w trakcie wykonywania, a nie kompilowany do kodu maszynowego przed wykonaniem.

3) Gdzie jest wykonywany JavaScript (klient czy serwer)?

JavaScript jest wykonywany po stronie klienta (w przeglądarce internetowej).

4) Dlaczego JavaScript to bezpieczne narzędzie?

JavaScript jest bezpieczne, ponieważ ma ograniczone uprawnienia do zasobów komputera użytkownika i działa tylko w przeglądarce internetowej.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ wykorzystuje definicje obiektów, metody i zdarzenia do opisu właściwości oraz zachowań elementów.

6) Wymień zalety JavaScript:

  • Jest łatwy do nauki i stosunkowo prosty w użyciu.
  • Odciąża serwery dzięki wykonywaniu kodu po stronie klienta.
  • Umożliwia tworzenie dynamicznych i interaktywnych stron internetowych.

7) Framework to:

Framework to narzędzie lub zestaw bibliotek, które upraszczają tworzenie aplikacji, np. React, Angular.

8) Do aplikacji mobilnych można użyć frameworka:

Do aplikacji mobilnych można użyć frameworka React Native.

9) Aplikacje responsywne to:

Aplikacje dostosowujące się automatycznie do różnych rozmiarów ekranów i urządzeń.

10) Wytłumaczenie kodu:

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

Tworzy przycisk, który po kliknięciu wywołuje funkcję WinOpen() otwierającą nowe okno.

11) Wytłumaczenie kodu:

window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Otwiera nowe okno z plikiem obraz.html. Parametry sterują wyglądem okna:

  • toolbar=no – ukrywa pasek narzędzi.
  • directories=no – ukrywa katalogi.
  • menubar=no – ukrywa pasek menu.
  • height i width – ustalają wymiary okna.
  • top i left – określają pozycję okna na ekranie.

12) Opis instrukcji window.close():

Instrukcja window.close() zamyka bieżące okno przeglądarki.
